Remote-sensing image: platform construction type

(MARC21 007/06 values for remote-sensing images expressed using RDF)

Type of construction of the platform serving as the base for the remote-sensing device. For the purposes of this data element, "platform" refers to any structure that serves as a base, not only flat surfaces.
