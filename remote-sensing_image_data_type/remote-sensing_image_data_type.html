Remote-sensing image: data type

(MARC21 007/09-10 values for remote-sensing images expressed using RDF)

Spectral, acoustic, or magnetic characteristics of the data received by the device producing the remote-sensing image. It can be used to indicate both the wave length of radiation measured and the type of sensor used to measure it.
